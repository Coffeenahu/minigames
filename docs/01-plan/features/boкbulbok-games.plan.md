# Plan: 복불복 게임 모음 (boкbulbok-games)

**Date**: 2026-03-05
**Level**: Starter → Dynamic (React)
**Status**: Plan

---

## 1. 개요 (Overview)

여러 명이 함께 즐길 수 있는 복불복 미니게임 모음 웹앱.
4가지 게임을 하나의 React 앱으로 제공하며, 참가자 이름을 입력하면 랜덤하게 결과를 배정한다.

---

## 2. 목표 (Goals)

- 웹 브라우저에서 바로 실행 가능한 복불복 게임 모음
- 4가지 게임 수록: 사다리 타기, 동전던지기, 해적룰렛게임, 주사위던지기
- 시각적으로 재미있는 애니메이션 포함
- 반응형 UI (모바일/PC 모두 지원)
- **게임 추가 확장성**: 새 게임을 파일 하나 추가하는 것만으로 홈 화면에 자동 등록되는 구조

---

## 3. 게임 목록 및 기능 정의

### 3.1 사다리 타기 (Ladder Game)

| 항목 | 내용 |
|------|------|
| 참가자 수 | 2~8명 |
| 입력 | 참가자 이름, 결과 항목 (당첨/꽝 등 커스텀) |
| 동작 | Canvas 또는 SVG로 사다리 렌더링, 애니메이션으로 경로 추적 |
| 결과 | 각 참가자가 어떤 결과에 연결되는지 표시 |

### 3.2 동전던지기 (Coin Flip)

| 항목 | 내용 |
|------|------|
| 참가자 수 | 1~N명 |
| 입력 | (선택) 참가자 이름 |
| 동작 | 3D CSS 코인 플립 애니메이션 |
| 결과 | 앞면(Head) / 뒷면(Tail) 표시 |

### 3.3 해적룰렛게임 (Pirate Roulette)

| 항목 | 내용 |
|------|------|
| 참가자 수 | 2~6명 |
| 입력 | 참가자 이름 (순서대로 칼 꽂기) |
| 동작 | 해적 배럴 이미지, 칼 하나를 꽂을 때마다 랜덤으로 배럴이 튀어오를지 결정 (마지막 한 명이 무조건 탈락) |
| 결과 | 배럴 폭발 애니메이션 + 탈락자 표시 |

### 3.4 주사위던지기 (Dice Roll)

| 항목 | 내용 |
|------|------|
| 참가자 수 | 1~N명 |
| 입력 | (선택) 참가자 이름, 주사위 개수 (1~3개) |
| 동작 | 3D CSS 주사위 굴리기 애니메이션 |
| 결과 | 각 참가자 주사위 결과, 최고/최저 점수자 강조 |

## 4. 기술 스택 (Tech Stack)

| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | React 18 | 컴포넌트 분리, 상태 관리 용이 |
| 빌드 도구 | Vite | 빠른 개발 서버, 간단한 설정 |
| 스타일 | CSS Modules 또는 Tailwind CSS | 게임별 독립 스타일 |
| 라우팅 | React Router v6 | 게임 선택 화면 → 각 게임 페이지 |
| 애니메이션 | CSS Animation / Framer Motion | 코인, 주사위 3D 효과 |
| 상태 관리 | React useState / useContext | 참가자 정보 전역 공유 |

## 5. 페이지 구조

```
/                   → 메인 홈 (게임 선택 화면)
/ladder             → 사다리 타기
/coin               → 동전던지기
/pirate             → 해적룰렛게임
/dice               → 주사위던지기
```

## 6. 프로젝트 디렉토리 구조

```
minigames/
├── public/
│   └── assets/         # 이미지, 아이콘
├── src/
│   ├── games/          # 게임별 독립 모듈 (확장 핵심 디렉토리)
│   │   ├── ladder/     # 사다리 타기
│   │   ├── coin/       # 동전던지기
│   │   ├── pirate/     # 해적룰렛
│   │   ├── dice/       # 주사위
│   │   └── index.js    # 게임 레지스트리 (새 게임은 여기에만 등록)
│   ├── components/
│   │   └── common/     # 공통 컴포넌트 (Header, Button, PlayerInput, GameCard)
│   ├── pages/
│   │   ├── Home.jsx    # 게임 레지스트리를 읽어 자동으로 카드 렌더링
│   │   └── GamePage.jsx # 모든 게임이 공유하는 단일 게임 페이지 래퍼
│   ├── App.jsx
│   └── main.jsx
├── package.json
└── vite.config.js
```

### 게임 레지스트리 패턴 (확장 방법)

새 게임 추가 시 `src/games/index.js`에 항목만 추가하면 홈 화면 카드와 라우트가 자동 생성:

```js
// src/games/index.js
export const GAMES = [
  { id: 'coin',   name: '동전던지기',   icon: '🪙', path: '/coin',   component: () => import('./coin/CoinGame') },
  { id: 'dice',   name: '주사위던지기', icon: '🎲', path: '/dice',   component: () => import('./dice/DiceGame') },
  { id: 'pirate', name: '해적룰렛',     icon: '☠️', path: '/pirate', component: () => import('./pirate/PirateGame') },
  { id: 'ladder', name: '사다리 타기',  icon: '🪜', path: '/ladder', component: () => import('./ladder/LadderGame') },
  // 새 게임 추가: 이 줄만 추가하면 끝
];
```
