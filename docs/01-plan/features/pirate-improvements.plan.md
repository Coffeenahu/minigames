# Plan: 해적룰렛게임 (Pirate) 개선

**Date**: 2026-03-05  
**Status**: Plan  
**Priority**: Medium  

---

## 1. 개요 (Overview)

현재 해적룰렛게임의 게임플레이, 그래픽, UI/UX를 개선하여 더욱 재미있고 사실적인 게임 경험을 제공한다.

---

## 2. 수정사항 (Improvements)

### 2.1 게임플레이 조정

**현황**
- 슬롯 개수: 참가자 수 × 2 (예: 4명 → 8개 슬롯)

**요구사항**
- 슬롯 개수: 참가자 수 × 4로 변경 (예: 4명 → 16개 슬롯)

**기대 효과**
- 게임이 더 오래 진행되어 긴장감 증대
- 플레이어들이 더 많은 턴을 경험할 수 있음
- 실제 해적룰렛과 더 유사한 경험 제공

**변경 코드 위치**
- `src/games/pirate/PirateGame.jsx` - `initGame()` 함수
  ```javascript
  // 현재
  const totalSlots = players.length * 2;
  
  // 변경
  const totalSlots = players.length * 4;
  ```

### 2.2 그래픽 개선

**현황**
- 해적 캐릭터와 나무통이 CSS로 그려진 추상적인 도형
- 실제 해적룰렛 게임의 외형과 거리가 있음

**요구사항**
- 실제 해적룰렛 게임과 유사한 그래픽 구현
- 나무통: 더 사실적인 통 모양 (목재 텍스처, 금속 테두리 등)
- 해적: 더 귀엽고 명확한 캐릭터 표현

**구현 방향**
1. **나무통 개선**
   - 원형 통 모양 강화 (원통형 시각 효과)
   - 목재 패턴 또는 그라데이션 추가
   - 칼 꽂힐 부분 명확화 (구멍 표시)
   - 폭발 시 파편 효과 추가

2. **해적 캐릭터 개선**
   - 더 큰 눈, 명확한 해적 모자
   - 간단하고 귀여운 얼굴 표현
   - 점프 시 몸 전체 움직임 강화

3. **색상 및 그림자**
   - 입체감 있는 그라데이션과 그림자 추가
   - 폭발 이펙트 강화 (파티클 효과 고려)

**변경 파일**
- `src/games/pirate/PirateGame.module.css` - 스타일 전면 개선
- `src/games/pirate/PirateGame.jsx` - 구조 최적화 (필요시)

---

### 2.3 UI/UX 개선

**현황**
- 슬롯 버튼이 단순한 사각형 형태

**요구사항**
- 슬롯 버튼이 실제 나무통에 칼을 꽂는 구멍처럼 보이도록 디자인
- 더욱 직관적이고 시각적인 상호작용

**구현 방향**
1. **슬롯 버튼 디자인**
   - 원형 또는 타원형 구멍 모양
   - 나무통 텍스처와 어울리는 색상
   - 호버/활성 상태 시각화
   - 칼이 꽂힌 상태 표현 개선

2. **상호작용 피드백**
   - 칼이 꽂힐 때 부드러운 애니메이션
   - 구멍에 칼이 박히는 느낌
   - 비활성 슬롯 (이미 칼이 꽂힌 것) 명확하게 표시

3. **레이아웃 조정**
   - 슬롯을 그리드로 배치 (4배수 변경에 맞춰)
   - 더 강화된 시각적 계층 구조

**변경 파일**
- `src/games/pirate/PirateGame.module.css` - 슬롯 스타일 재설계
- `src/games/pirate/PirateGame.jsx` - 그리드 레이아웃 조정
