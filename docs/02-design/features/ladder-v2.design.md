# [DESIGN] 사다리 게임 V2 개편 디자인

## 1. 개요
Stitch v2 디자인 에셋을 기반으로 사다리 게임의 사용자 경험(UX)과 인터페이스(UI)를 현대적이고 세련된 스타일로 전면 개편합니다.

## 2. 디자인 원칙
- **색상 체계:** 파스텔 톤의 보라색과 연한 핑크색을 주축으로 함.
  - Primary: `#9F9FED` (보라색)
  - Secondary: `#D4C1EC` (연한 보라색)
  - Background: `#FEF9FF` (매우 연한 핑크)
- **타이포그래피:** 'Plus Jakarta Sans' 폰트를 사용하여 모던하고 깔끔한 느낌 전달.
- **컴포넌트 스타일:** 둥근 모서리(Rounded), 부드러운 그림자(Shadow), 카드 형태의 레이아웃.

## 3. 화면별 설계

### 3.1. 1단계: 참여자 명단 입력 (`home.html` 참조)
- **헤더:** 뒤로가기 버튼과 "참여자 명단 입력" 타이틀.
- **입력 목록:** 각 참여자는 번호 아이콘, 입력 필드, 삭제 버튼이 포함된 가로 카드 형태로 표시.
- **추가 버튼:** 점선 테두리가 있는 "인원 추가하기" 버튼.
- **하단 버튼:** "다음으로 (1/2)" 고정 버튼.

### 3.2. 2단계: 결과 입력 (`ladder_setup.html` 참조)
- **헤더:** "사다리 설정 2단계" 타이틀.
- **입력 필드:** 참여자 수만큼의 결과 입력 필드 제공. (사용자 편의를 위해 기본 '통과' 안내 텍스트 또는 빈 칸 유지)
- **동적 제어:** 참여자 수와 결과의 수가 일치하도록 제어하며, 최대 인원을 초과할 수 없음.
- **자동 완성:** 입력되지 않은 빈 필드는 "통과"로 자동 처리됨을 사용자에게 안내.
- **하단 버튼:** "이전으로"와 "게임 시작하기 (2/2)" 버튼.

### 3.3. 3단계: 사다리 게임 진행 (`ladder_game.html` 참조)
- **상단:** 참여자 이름 표시.
- **사다리 영역:** `ladderUtils.js`로 생성된 로직을 SVG 또는 Canvas로 렌더링.
- **하단 결과 영역:** 사다리 하단에 아이콘(선물, 꽝 등)과 결과 레이블 표시.
- **액션:** "전체 결과 보기" 및 "다시 하기" 버튼.

### 3.4. 4단계: 결과 팝업 (`ladder_result.html` 참조)
- **오버레이:** 블러(Blur) 처리된 배경 위에 모달 창 표시.
- **내용:** "축하합니다!" 메시지, 당첨자 이름, 결과 텍스트, 축하 아이콘(celebration).
- **액션:** "확인" 버튼을 통해 팝업 닫기.

## 4. 데이터 모델
- `Step`: `PLAYERS` | `RESULTS` | `GAME`
- `Player`: `{ id: number, name: string }`
- `Outcome`: `{ id: number, text: string }`
- `LadderData`: 사다리 선(Line) 및 교차로(Step) 정보.

## 5. 애니메이션 및 인터랙션
- **경로 추적:** 사용자가 참여자를 클릭하거나 "전체 보기" 시, 경로가 색상별로 부드럽게 그려지는 효과.
- **전환 효과:** 단계 이동 시 좌우 슬라이드 또는 페이드 효과 적용 고려.
